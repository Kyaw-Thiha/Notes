## Convex Set
> `Defn`: A set $C$ in a vector space is `convex` if, 
> for any two vectors $\mathbf{u}, \mathbf{v} \in C$, 
> the line segment between $\mathbf{u}$ and $\mathbf{v}$ is contained in $C$.

![Convex Set|400](https://i.ytimg.com/vi/oX_WK5co4HY/maxresdefault.jpg)

That is for any $\alpha \in [0, 1]$, then $\alpha \ \mathbf{u} + (1 - \alpha) \ \mathbf{v} \in C$.

In other words, given an $\alpha \in [0, 1]$, the point $\alpha \ \mathbf{u} + (1 - \alpha) \ \mathbf{v}$ is a `convex combination`.
