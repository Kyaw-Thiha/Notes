
## Convex Set
`Defn`: A set $C$ in a vector space is `convex` if, for any two vectors $\mathbf{u}, \mathbf{v} \in C$, the line segment between $\mathbf{u}$ and $\mathbf{v}$ is contained in $C$.
That is for any $\alpha \in [0, 1]$, then $\alpha \ \mathbf{u} + (1 - \alpha) \ \mathbf{v} \in C$.
