# Monocular Depth Estimation
`Monocular Depth Estimation` estimate the the depth given a single image purely on appearance cues.

![Monocular Depth Estimation|300](https://images.ctfassets.net/x1jx2zz0m6ly/6zqQXxYIefQQ49aPRy0rTB/38188b7df1eff35889e022073b5d3e8a/20211110_Blog_gif.gif)

Note that `depth estimation` is fundamentally **ill-posed** since infinitely many 3D scenes can project to the same 2D image.
