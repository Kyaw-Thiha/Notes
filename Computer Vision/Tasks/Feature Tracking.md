# Feature Tracking
Given a set of points/regions in frame $t$, `feature tracking` find their corresponding locations in frame $t+1$.

## Where feature tracking is used
`Feature tracking` is more of an enabling primitive than an actual end product.

Its use cases are as follow
- **Visual SLAM** and **sfM**
- **4D Dynamic Scene Reconstruction**
- **Video Understanding** and **Action Recognition**
- **Augmented Reality**
- **Video Editing** and **VFX**
