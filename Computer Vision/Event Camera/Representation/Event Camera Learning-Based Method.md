## Learning-Based Method
> **Learn** the event representation end-to-end rather than hand-crafting it.
> This enable task-agnostic, flexible representations optimized for specific downstream objectives

## Key Methods
- **MLP-based Representation**
  First learning-based approach for event-to-tensor conversion.
- **LSTM-based Representation**
  Events are sequential. 
  So, recurrent models capture temporal dependencies.
