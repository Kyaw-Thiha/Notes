# Banach Fixed Point Theorem
#math 
If $L: \mathcal{Z} \to \mathcal{Z}$ is a [[Contraction Mapping|contraction mapping]] on a complete [[Metric|metric space]] $(Z, d)$, then there exists a unique $z^{*} \in \mathcal{Z}$ such that $Lz^{*} = z^{*}$.

Furthermore, the point $z^{*}$ can be found by choosing an arbitrary $z_{0} \in \mathcal{Z}$ and defining $z_{k+1} = L \ z_{k}$.
Then, we will have $z_{k} \to z^{*}$.
