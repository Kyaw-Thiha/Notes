# Fixed Point
#math 
If $L: \mathcal{Z} \to \mathcal{Z}$, then a point $z \in \mathcal{Z}$ such that 
$$
Lz  = z
$$
is called a `fixed point` of $L$.

Given an equation $f(z) = 0$, we can convert it to a fixed point equation $Lz = z$ by defining
$$
L: z \mapsto f(z) + z
$$
Then if $Lz^{*} = z^{*}$ for a $z^{*}$, we get that $f(z^{*}) = 0$.
The fixed point of $L$ is the same as the solution of $f(z) = 0$.
