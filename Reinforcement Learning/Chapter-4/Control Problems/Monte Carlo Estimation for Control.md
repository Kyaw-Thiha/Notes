# Monte Carlo Estimation
We can use similar methods to  [[Monte Carlo Estimation for Policy Evaluation]] and [[Temporal Difference Learning for Policy Evaluation(TD)|TD Learning]], to solve the control problem.

The general idea is to use some version of [[Policy Iteration]].
If we run many rollouts from each state-action pair $(s,a)$, we can define $\hat{Q}^{\pi}_{t}$ that converges to $Q^{\pi}$.

If we wait for infinite time,
$$
\hat{Q}^{\pi}_{\infty} 
= \lim_{ t \to \infty } \hat{Q}^{\pi}_{t}
= Q^{\pi}
$$
We can then choose $\pi' \leftarrow \pi_{g}(\hat{Q}^{\pi}_{\infty})$.

This [[Policy Iteration]] can be described by the following sequence of $\pi$ and $Q^{\pi}$:
$$
\pi_{0} \xrightarrow{\text{ E }} Q^{\pi_{0}}
\xrightarrow{\text{ I }}
\pi_{1} \xrightarrow{\text{ E }} Q^{\pi_{1}}
\xrightarrow{\text{ I }} \dots
$$
Note that 
- We do need to have a very accurate estimation of $Q^{\pi_{k}}$ before performing the [[Policy Iteration|policy improvement step]].
- We can perform Monte-Carlo estimation for a finite number of rollouts from each state, and then perform the [[Policy Iteration|improvement step]].

---
## Monte Carlo Control (Initial State Only)
> **Require:** Initial policy $\pi_1$, step size schedule $(\alpha_k)_{k \geq 1}$.  
> Initialize $\hat{Q}_1 : \mathcal{S} \times \mathcal{A} \to \mathbb{R}$ arbitrary, e.g., $\hat{Q}_1 = 0$.

>- **for** each iteration $k = 1, 2, \dots$ **do**
>  - **for** all $(s, a) \in \mathcal{S} \times \mathcal{A}$ **do**
>    - Initialize $S_1 = s$ and $A_1 = a$
>    - Generate an episode from $S_1$ by choosing $A_1$, then following $\pi_k$ to obtain $S_1, A_1, R_1, S_2, A_2, R_2, \dots$
>    - Compute $G_1^{\pi_k}(S_1, A_1) = \sum_{t \geq 1} \gamma^{t-1} R_t$
>    - Update

$$\hat{Q}_{k+1}(S_1, A_1) \leftarrow \bigl(1 - \alpha_k(S_1, A_1)\bigr)\hat{Q}_k(S_1, A_1) + \alpha_k(S_1, A_1)\, G_1^{\pi_k}(S_1, A_1)$$

>  - **end for**
>  - Improve policy: $\pi_{k+1} \leftarrow \pi_g(Q_{k+1})$
>- **end for**

---
### Convergence of MC for Control
The sequence $Q_{k}$ generated by the MC control algorithm with a learning rate $\alpha_{k}$ satisfying the [[Stochastic Approximation(SA)|SA conditions]] converge to $Q^{*}$ almost surely.

$$
\sum ^{\infty}_{t=0} \alpha_{t} = \infty, 
\ \quad \ 
\sum ^{\infty}_{t=0} \alpha^{2}_{t}(s) < \infty
$$
