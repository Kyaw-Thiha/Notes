# Parametric vs Non-Parametric Models

**Parametric Model:** No. of parameters does not change based on the dataset size
**Non-Parametric model**: No. of parameters change based on dataset size

## Parametric Models
- Linear Regression
- Logistic Regression
- Naive Bayes

## Non-Parametric Models

- [[K-NN]]
- Decision Trees
- Random Forest
- Kernal Methods (SVM)
