# Expected Target Value

Due to `noise` of the data, there maybe different values of `y` for a given value of `x`.


![[Expected Target Value.png]]

The `Expected Target Value` is denoted as 
$$
\bar{y}(x) = E_{Y|X}[Y] = \int y.p(y|x) \ dx
$$

In other words, it is the mean of all values `y` for the given value of `x`.
