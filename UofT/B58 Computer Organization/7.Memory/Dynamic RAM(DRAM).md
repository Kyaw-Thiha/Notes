# DRAM
> [[Capacitor|Capacitor]] charge state indicates stored value.
> Whether the [[Capacitor|capacitor]] is charged or discharged indicates a storage value of `1` and `0` respectively

It consists of
- $1$ [[Capacitor]]
- $1$ [[Transistor|Access Transistor]]

[[Capacitor]] leaks through the `RC Path`, so
- [[Dynamic RAM(DRAM)|DRAM cell]] loses charge over time.
- [[Dynamic RAM(DRAM)|DRAM cell]] needs to be refreshed.
