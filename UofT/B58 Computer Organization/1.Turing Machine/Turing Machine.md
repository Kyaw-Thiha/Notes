# Turing Machine
A [[Turing Machine]] is a theoretical, abstract model of computation that help defines the limits of algorithmic calculation.
![Turing Machine|200](https://upload.wikimedia.org/wikipedia/commons/thumb/a/ad/Model_of_a_Turing_machine.jpg/250px-Model_of_a_Turing_machine.jpg)

---
## Components of Turing Machine
[[Turing Machine]] has an infinite tape, a read/write head, and rules that control how it reads, writes and moves on the tape.

---
## Behavior of Turing Machine
- The behaviour of a [[Turing Machine]] is controlled by [[Deterministic Finite State Automata|finite state machine]] consisting of a finite set of states, a transition function, and start and accept states.
- The [[Turing Machine|machine]] starts in the initial state, and follows the transition rules until it reaches an accept or reject state.

---
## How it works
A [[Turing Machine]] consists of a tape of infinite length on which read and write operations can be performed.

![Turing Machine|300](https://media.geeksforgeeks.org/wp-content/uploads/20260108120340749793/tape.webp)

The tape consists of infinite cells on which each cell contain input symbol(alphabet) or a special symbol called blank.

It also consists of a head pointer which points to cell currently being read, and it can move in both directions.
