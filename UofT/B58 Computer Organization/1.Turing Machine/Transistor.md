# Transistor

## Bipolar Junction Transistor (BJT)
A [[Transistor|bipolar junction transistor]] works by using a small current to control a larger current.

![image|500](https://notes-media.kthiha.com/Transistor/f185e174ee10254dd2818bde8972e669.png)

It has $3$ different terminals: 
- **Base**: The control input (where current is applied)
- **Collector**: Where current flows in
- **Emitter**: Where current flows out

---
## MOSFET
A [[Transistor|metal-oxide semiconductor field-effect transistor]] uses voltage(electric field) instead of current to control a larger current.

![image|500](https://notes-media.kthiha.com/Transistor/52201bf87827cb03248a140c0ba7c777.png)

Its $3$ terminals are:
- **Gate**: the control input (where voltage is applied)
- **Drain**: where current flows in
- **Source**: where current flows out

The gate is insulated from the rest of the device by a thin oxide layer. So, it draws almost no current at all, making it more efficient than [[Transistor|BJT]].
