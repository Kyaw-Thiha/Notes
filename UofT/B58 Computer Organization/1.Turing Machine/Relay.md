# Relay
A relay uses an electromagnet(a coil of wire) to mechanically pull or push a switch. This opens or closes a circuit.
![Relay|300](https://www.build-electronic-circuits.com/wp-content/uploads/2024/02/RELAY_NO_2-1.gif)

---
### Building Logic Gates
Relays can be can be stacked or arranged together to build different logic gates.
- Let `NC` stands for normally closed.
- And let `NO` stands for normally opened.
