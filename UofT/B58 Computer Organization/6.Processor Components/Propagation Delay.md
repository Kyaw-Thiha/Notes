# Propagation Delay
> [[Propagation Delay|Propagation delay]] is the maximum time from when an input changes until the output or outputs reach their final value.

![Propagation Delay|250](https://ocw.mit.edu/courses/6-004-computation-structures-spring-2017/32605907e13d9fe3f9daf48e21474a70_Slide15.png)

---
## Calculating Propagation Delay
- Find the `critical path`(path with largest number of gates)
- Sum up the [[Propagation Delay|propagation delay]] of all gates on the critical path

---
[[Propagation Delay|Propagation delay]] should typically upper-bounded. 
This can be done by shortening the `critical path`.
