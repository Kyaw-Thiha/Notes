# Ripple-Carry Binary Adder
Given that a [[Full Adder|full adder]] accepts a carry-bit and outputs a carry-bit, we can chain them up to form [[Ripple-Carry Binary Adder]] in order to add multiple bits.

![image|350](https://notes-media.kthiha.com/Ripple-Carry-Binary-Adder/fb0a1fc5ac9ed974eb28b60dd212d66b.png)
where $S_{3}S_{2}S_{1}S_{0}$ is the sum of $X_{3}X_{2}X_{1}X_{0}$ and $Y_{3}Y_{2}Y_{1}Y_{0}$.
