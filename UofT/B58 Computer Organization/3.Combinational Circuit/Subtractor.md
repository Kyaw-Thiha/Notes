# Subtractor
[[Subtractor]] performs addition on negative numbers.

---
## Circuit
We can use the first $C_{in}$ of [[Ripple-Carry Binary Adder|ripple-carry binary adder]] as the $1$ being added for [[Two's Complement|2's complement]].
![image|350](https://notes-media.kthiha.com/Subtractor/d94a3600214d1ee0fca914525fbd37ae.png)

Hence, we can use one [[Ripple-Carry Binary Adder|circuit]] as both an [[Adder|adder]] and a [[Subtractor|subtractor]].
