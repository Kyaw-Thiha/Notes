# Intersection of 2 AVL Trees
Given $T$ and $V$, we want an [[AVL Tree]] containing the keys in both $T$ and $V$.
![image|300](https://notes-media.kthiha.com/Intersection-of-AVL-Trees/4621cbeab83c507f5fb4cde302956eb0.png)

Note that this where $b$ from split is used.
If $b$ is true, that means both $T$ and $V$ has it so it is returned.
Otherwise, only $V$ has it so it's not returned.
